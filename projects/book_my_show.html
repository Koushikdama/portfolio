book_my_show

  • Tech Stack: Django, HTML, CSS, JavaScript, Bootstrap, SQLite, Pandas, vercel
  • Github URL: Project Link

Description The Book My Show Clone App allows users to book, cancel, and manage movie tickets, featuring dynamic pricing based on seat availability and user preferences. Built with Django for backend development, Django Channels for handling real-time communication, and Signals to manage important events (e.g., ticket booking confirmation). The frontend is built using HTML, CSS, and JavaScript to create a responsive and engaging user interface. Users receive email notifications upon ticket booking and cancellation, and dynamic pricing ensures that ticket prices adjust based on availability and demand.

Features

  • User Authentication

    • Users can create accounts, log in, and manage their profiles using Django's authentication system.

  • Movie Listings and Show Timings

    • Users can view a list of available movies, along with show timings, theaters, and seat availability.

  • Dynamic Ticket Pricing

    • Prices for movie tickets vary depending on factors such as seat location, movie popularity, and timing. Dynamic pricing adjusts ticket cost in real time.

  • Ticket Booking

    • Users can select a movie, show timing, and available seats to book tickets. They can add the tickets to the cart and proceed to checkout.

  • Ticket Cancellation

    • Users have the option to cancel their booked tickets. Upon cancellation, the system updates seat availability, and users are notified.

  • Real-time Updates with Django Channels

    • Django Channels enables real-time updates such as seat availability, ticket booking confirmations, and dynamic price changes during the booking process.
